# README

## Getting Started

### clone repository

```
$ cd /to/your/path
$ git clone git@github.com:edwardkenfox/burn-me-down.git
```

### install gems

```
$ bundle install
```

### create database 

```
$ rake db:create
```

### start server

```
$ rails s
```

### done!
